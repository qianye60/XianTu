# NPC命名生成规范 - 独特性保障指南

## 🎯 核心使命
**每个NPC的名字必须是唯一的、有意义的、与角色灵魂契合的标识**

---

## 📋 强制执行规则

### Rule 1: 禁止直接使用示例名字
- ❌ 严禁使用任何文档中出现的示例名字(如:楚清璃、墨临渊、剑无尘等)
- ❌ 严禁使用过于相似的变体(如:楚清玉、墨临川)
- ✅ 必须基于角色背景从零创造

### Rule 2: 名字必须服务于角色灵魂
命名顺序: **角色核心驱动力 → 背景经历 → 命名风格 → 具体名字**

```
错误流程: "需要一个剑修 → 取名'剑无尘'"
正确流程: "复仇者(核心) → 灭门惨案(经历) → 冷硬孤绝(气质) → 寒骨"
```

### Rule 3: 同场景NPC命名差异化
- 同一情境出现的多个NPC,必须使用不同的命名策略
- 避免"复姓扎堆"或"单字名集中"
- 确保读者/玩家能通过名字快速区分角色

---

## 🏗️ 命名构建流程

### Step 1: 解析角色核心
在命名前必须明确:
- **核心驱动力**: 角色最深层的欲望/恐惧/信念
- **关键经历**: 塑造性格的转折事件
- **身份标签**: 出身、门派、种族(但不是决定因素)

### Step 2: 提取命名意象
从角色核心中提取1-2个**具象意象**:
- 守护者 → 盾、壁、岳、镇
- 复仇者 → 刃、霜、破、绝
- 求道者 → 虚、明、证、悟
- 逃避者 → 影、幽、隐、尘

### Step 3: 选择命名结构
根据角色的社会地位和性格选择结构:

| 角色类型 | 推荐结构 | 气质特点 |
|---------|---------|---------|
| 世家贵族 | 复姓+双字 | 雍容华贵,门第森严 |
| 宗门长老 | 单姓+道号 | 超然物外,修道有成 |
| 天才弟子 | 单姓+双字 | 英姿勃发,名扬四海 |
| 江湖散修 | 单姓+单字 | 简洁刚硬,不拘一格 |
| 神秘高人 | 道号/绰号 | 身份成谜,威名远播 |

### Step 4: 组合生成名字
**动态组合**而非**模板套用**:

```
示例: 角色=炼丹宗师,痴迷禁术,师门被灭
↓
提取意象: 火、丹、执念、孤绝
↓
选择结构: 单姓+双字(强调个人执念)
↓
候选组合: 
- 姓(炽/焰/烬) + 名(药痴/丹执/孤焚)
- 姓(萧/慕/宁) + 名(火道/烬心)
↓
最终选择: 宁烬心(宁静的外表,焚烧的内心)
```

## 🎨 命名意象素材库(非固定模板)

### 自然元素类
**天象**: 云、霜、雪、雷、风、雨、星、月、日、辰、曦、暮
**地貌**: 岳、渊、峰、谷、海、川、林、泽、岩、崖
**季节**: 春、秋、寒、暖、霖、霁

### 修道意蕴类
**境界**: 虚、明、空、寂、证、悟、灵、玄、妙、真
**心性**: 清、净、澄、素、淡、朴、拙、愚、痴
**道器**: 剑、琴、棋、书、丹、符、阵、印

### 性格气质类
**刚硬**: 铁、石、钢、磐、岳、戟、破、烈
**柔和**: 柳、绢、绮、锦、瑟、箫、婉、柔
**孤傲**: 独、孤、寒、冷、霜、雪、绝、孑

### 时代痕迹类
**古韵**: 之、若、如、兮、焉、然
**新意**: 逸、羽、辰、宸、煜、琰

## 🔄 命名风格动态策略

### 策略A: 反差命名法
**打破刻板印象**:
- 火系修士不一定带"炎/焰/烈"
- 冷艳剑修不一定用"寒/冰/霜"
- 邪修不一定用"血/魔/煞"

**示例**:
- 性格暴烈的火修 → 取名"沈寂川"(以静制动)
- 冷面杀手 → 取名"温故言"(冷热反差)

### 策略B: 经历烙印法
**从关键事件提取命名要素**:
- 灭门幸存者 → "遗孤/孑/残/存"等字
- 奇遇得道者 → "缘/遇/奇/幸"等字
- 师恩铭记者 → 师父名字的某个字

### 策略C: 文化根植法
**根据地域/门派文化命名**:
- 北疆蛮荒 → 简短有力(单字名为主)
- 中州世家 → 文雅繁复(复姓+双字)
- 西域妖族 → 异域特色(罕见字)

### 策略D: 隐喻象征法
**用名字暗示角色命运/性格**:
- 外冷内热 → 表面冰冷的字+温暖的字
- 表里不一 → 正面字+负面意蕴
- 矛盾挣扎 → 对立意象组合

## ✅ 命名质量检查清单

在确认名字前,必须回答以下问题:

1. [ ] 这个名字是否与角色的核心驱动力契合?
2. [ ] 是否避免了使用示例名字或高度相似的变体?
3. [ ] 是否与已登场的NPC名字有明显区分?
4. [ ] 名字是否体现了角色独特的背景经历?
5. [ ] 是否打破了性别/门派/种族的刻板印象?
6. [ ] 读起来是否自然,不会过于生僻或拗口?
7. [ ] 是否避免了过度使用常见姓氏(张王李赵刘)?

## 🚫 反面教材 - 常见错误

### 错误1: 模板化套用
```
❌ "火系女修 → 自动命名'烈焰仙子'"
✅ "火系女修 → 分析角色核心 → 可能是'沈霁月'(心如死灰的复仇者)"
```

### 错误2: 示例名直接复用
```
❌ "需要剑修 → 使用示例'剑无尘'"
✅ "需要剑修 → 分析背景 → 创造'秋霜剑'(秋日考核失败,从此苦修)"
```

### 错误3: 缺乏角色关联
```
❌ "随机组合字库 → '欧阳星辰'(与角色无关)"
✅ "观星预言者 → '司星观'(职业即名字)"
```

### 错误4: 同场景重复风格
```
❌ 同时出现"司徒云锦""欧阳星河""南宫雪霜"(复姓扎堆)
✅ 混合使用"司徒云锦""宁寒""玄虚子"(多样化)
```

## 💡 高级技巧

### 技巧1: 双重意蕴命名
名字表面一层含义,深层另一层含义:
- "清和"表面平和,实则"清算"+"和解"
- "慕容"表面优雅,实则"慕忠烈之容"

### 技巧2: 代际传承命名
师徒/父子/同门可共享某个字:
- 师父"青云道人" → 弟子"青XX""XX云"
- 避免完全相同,保持独立性

### 技巧3: 成长变化命名
角色可能有多个名字阶段:
- 幼名(乳名) → 正式名 → 道号 → 绰号
- 反映身份和心境的变化

## 📌 记住
**名字不是随机组合文字,而是角色灵魂的外化表达**
**每一个名字背后,都应该有一个能讲述的故事**
